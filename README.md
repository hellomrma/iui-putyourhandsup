# IUI Put Your Hands Up 🎵

IUI(인터랙션 컨퍼런스)를 위한 모던한 웹 음악 플레이어입니다. 비닐 레코드에서 영감을 받은 애니메이션 요소와 실시간 가사 표시, 반응형 디자인을 갖춘 미니멀하고 세련된 음악 플레이어입니다.

## 📋 프로젝트 개요

이 프로젝트는 IUI 컨퍼런스의 웹 개발 관련 주제를 다룬 음악 트랙들을 재생할 수 있는 인터랙티브 웹 플레이어입니다. 바닐라 JavaScript로 구현되었으며, 외부 의존성 없이 순수 HTML, CSS, JavaScript만으로 동작합니다.

## 🎯 주요 기능

### 🎵 모던한 미니멀 디자인
- **세련된 UI**: 블랙, 그레이, 화이트 컬러의 미니멀한 컬러 테마
- **동적 배경**: 떠다니는 구체(orb)와 회전하는 링(ring)으로 구성된 미래지향적 장식 요소
- **부드러운 전환 효과**: 모든 인터랙션에 페이드 및 트랜지션 애니메이션 적용
- **비닐 레코드 모티프**: 클래식한 비닐 레코드를 모던하게 재해석한 시각적 요소

### 🎧 비닐 레코드 모티프
- **회전 애니메이션**: 재생 중 지속적으로 회전하는 비닐 레코드 스타일 요소
- **바운스 효과**: 위아래로 부드럽게 움직이는 레코드 컨테이너
- **상태 표시**: 재생/일시정지 상태에 따라 애니메이션 제어
- **그루브 디테일**: 레코드 표면의 홈(groove) 패턴을 모던하게 시각화

### 📝 가사 표시 시스템
- **노래방 스타일**: 한 번에 3줄의 가사 표시 (위 1줄, 현재 줄, 아래 1줄)
- **활성 줄 강조**: 현재 가사 줄이 더 크고 밝게 표시되며 텍스트 그림자 효과
- **부드러운 전환**: 페이드인/아웃 효과로 가사 변경 시 자연스러운 전환
- **수동 탐색**: 키보드 화살표 키(↑/↓)로 가사 수동 탐색
- **터치 제스처**: 모바일에서 위/아래 스와이프로 가사 탐색

### ⌨️ 키보드 단축키
- **Space**: 재생/일시정지
- **← (Left Arrow)**: 이전 트랙
- **→ (Right Arrow)**: 다음 트랙
- **↑ (Up Arrow)**: 가사 위로 이동
- **↓ (Down Arrow)**: 가사 아래로 이동

### 🎛️ 플레이어 컨트롤
- **재생/일시정지**: 중앙의 큰 재생 버튼
- **이전/다음 트랙**: 좌우 네비게이션 버튼
- **진행 바**: 드래그하여 원하는 구간으로 이동
- **볼륨 조절**: 슬라이더로 볼륨 조절 (볼륨에 따라 아이콘 변경)
- **시간 표시**: 현재 시간 / 전체 시간 표시

### 📱 리모컨 플레이리스트
- **하단 고정**: 화면 하단에 고정된 가로 플레이리스트
- **트랙 정보**: 트랙 번호와 제목 표시
- **활성 트랙 강조**: 현재 재생 중인 트랙은 흰색 배경으로 강조
- **원클릭 선택**: 클릭 한 번으로 트랙 변경
- **반응형 레이아웃**: 모바일에서 한 줄에 2개 항목 표시

### 📱 반응형 디자인
- **데스크톱**: 플레이어와 가사 패널이 나란히 배치
- **태블릿 (1200px 이하)**: 세로 배치로 전환
- **모바일 (600px 이하)**: 
  - 레코드 판 숨김
  - 볼륨 컨트롤 숨김
  - 가사 폰트 크기 조정
  - 리모컨 플레이리스트 2열 레이아웃
  - 장식 요소 크기 축소

### 🎨 터치 제스처 지원
- **가사 스와이프**: 위/아래 스와이프로 가사 탐색 (최소 50px)
- **볼륨 슬라이더**: 터치 드래그로 볼륨 조절
- **진행 바**: 터치로 구간 이동

## 🚀 시작하기

### 요구사항
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- JavaScript 활성화
- 로컬 웹 서버 (선택사항, CORS 정책 때문에 권장)

### 설치 및 실행

1. **저장소 클론 또는 다운로드**
   ```bash
   git clone <repository-url>
   cd iui-putyourhandsup
   ```

2. **로컬 서버 실행 (권장)**
   
   Python 3가 설치되어 있다면:
   ```bash
   # Python 3
   python -m http.server 8000
   ```
   
   Node.js가 설치되어 있다면:
   ```bash
   # npx 사용
   npx http-server
   
   # 또는 http-server 글로벌 설치
   npm install -g http-server
   http-server
   ```
   
   또는 VS Code의 Live Server 확장을 사용할 수 있습니다.

3. **브라우저에서 열기**
   - 로컬 서버를 사용하는 경우: `http://localhost:8000`
   - 직접 열기: `index.html` 파일을 브라우저로 드래그 앤 드롭 (일부 기능 제한 가능)

## 🎵 음악 파일 추가하기

### 1. MP3 파일 추가
`music/` 폴더에 MP3 파일을 추가합니다.

### 2. 가사 파일 추가
같은 파일명으로 `.txt` 확장자를 가진 가사 파일을 생성합니다.
- 예: `my_song.mp3` → `my_song.txt`
- 가사 파일은 각 줄이 가사 한 줄을 나타냅니다.

### 3. 플레이리스트에 등록
`script.js` 파일의 `musicFiles` 배열을 수정합니다:

```javascript
const musicFiles = [
    "track1_재사용 만렙 컴포넌트, EJS에 심다 (feat. 김다인, 이미연, 박정인).mp3",
    "track2_Canvas 너머 3D 웹지피유, 어디까지 가봤니 (by 오지원).mp3",
    "my_song.mp3",  // 새 트랙 추가
    // 더 많은 트랙 추가...
];
```

### 4. 트랙 이름 형식 (선택사항)
트랙 이름이 `track{번호}_{제목}.mp3` 형식이면 리모컨 플레이리스트에서 번호와 제목이 자동으로 분리되어 표시됩니다.

## 🎨 커스터마이징

### 색상 테마 변경
`styles.css`에서 다음 변수들을 수정할 수 있습니다:
- 배경색: `body { background: #000000; }`
- 플레이어 배경: `.player { background: #1a1a1a; }`
- 텍스트 색상: `.track-title { color: #ffffff; }`

### 애니메이션 속도 조정
`styles.css`의 `@keyframes` 규칙에서 애니메이션 속도를 변경할 수 있습니다:
- 레코드 회전: `.vinyl-record.playing { animation: spin 3s linear infinite; }`
- 바운스 효과: `.vinyl-container { animation: bounce 2s ease-in-out infinite; }`

### 가사 표시 줄 수 변경
`script.js`의 `renderLyricsLines()` 함수에서 표시할 줄 수를 조정할 수 있습니다:
```javascript
const startIndex = Math.max(0, currentLyricIndex - 1);  // 위로 1줄
const endIndex = Math.min(lyricsLines.length, currentLyricIndex + 2);  // 아래로 1줄
```

### 반응형 중단점 조정
`styles.css`의 미디어 쿼리를 수정하여 반응형 중단점을 변경할 수 있습니다:
- 태블릿: `@media (max-width: 1200px)`
- 모바일: `@media (max-width: 600px)`

## 🎯 주요 기능 상세

### 레코드 판 애니메이션
- 재생 중: `spin` 애니메이션으로 지속 회전
- 일시정지: `animation-play-state: paused`로 회전 정지
- 바운스: 컨테이너가 위아래로 부드럽게 움직임
- 그루브: 4개의 동심원으로 레코드 홈 표현

### 가사 표시 시스템
- **자동 로드**: 트랙 변경 시 같은 이름의 `.txt` 파일 자동 로드
- **메모리 캐싱**: 한 번 로드한 가사는 `Map` 객체에 저장하여 재사용
- **3줄 표시**: 현재 줄 기준 위 1줄, 현재 줄, 아래 1줄
- **페이드 전환**: 가사 변경 시 150ms 페이드 아웃 후 페이드 인
- **인덱스 관리**: `currentLyricIndex`로 현재 활성 줄 추적

### 리모컨 플레이리스트
- **동적 렌더링**: `renderRemotePlaylist()` 함수로 트랙 목록 동적 생성
- **활성 상태**: `active` 클래스로 현재 트랙 시각적 구분
- **파일명 파싱**: 정규식으로 `track{번호}_{제목}` 형식 파싱
- **반응형**: 모바일에서 2열 그리드 레이아웃

### 배경 장식 요소
- **4개의 구체(Orb)**: 다양한 크기와 위치, 각각 다른 애니메이션 속도
- **3개의 링(Ring)**: 회전, 펄스, 드리프트, 글로우 애니메이션
- **애니메이션 조합**: float, pulse, drift, glow 등 여러 애니메이션 동시 적용
- **성능 최적화**: `pointer-events: none`으로 클릭 이벤트 차단

### 터치 제스처 처리
- **가사 스와이프**: `touchstart`, `touchend` 이벤트로 스와이프 감지
- **볼륨 슬라이더**: `touchstart`, `touchmove`, `touchend`로 드래그 처리
- **최소 거리**: 50px 이상 스와이프해야 가사 이동 (의도하지 않은 스크롤 방지)
