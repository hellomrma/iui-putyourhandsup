# Music Player - 쥬크박스 스타일

애니메이션 레코드 판, 가사 표시, 반응형 디자인을 갖춘 현대적인 쥬크박스 스타일 웹 음악 플레이어입니다.

## 주요 기능

- 🎵 **쥬크박스 디자인**: 애니메이션 배경 오브제가 있는 레트로-미래지향적인 블랙, 그레이, 화이트 테마
- 🎧 **애니메이션 레코드 판**: 바운스 애니메이션이 있는 회전하는 레코드 플레이어
- 📝 **가사 표시**: 노래방 스타일 형식으로 가사 표시 (한 번에 3줄)
- ⌨️ **키보드 네비게이션**: 화살표 키(↑/↓)를 사용하여 가사를 수동으로 탐색
- 📱 **반응형 디자인**: 데스크톱과 모바일 기기에 최적화
- 🎛️ **리모컨 플레이리스트**: 화면 하단의 가로 플레이리스트
- 🎨 **부드러운 애니메이션**: 떠다니는 구체, 회전하는 링, 부드러운 전환 효과

## 파일 구조

```text
.
├── index.html          # 메인 HTML 파일
├── styles.css          # CSS 스타일 및 애니메이션
├── script.js           # JavaScript 기능
├── music/              # 음악 파일 디렉토리
│   ├── *.mp3          # 오디오 파일
│   └── *.txt          # 가사 파일 (MP3와 같은 이름)
└── README.md          # 이 파일
```

## 설정 방법

1. 이 저장소를 클론하거나 다운로드합니다
2. MP3 파일을 `music/` 폴더에 넣습니다
3. 같은 파일명으로 `.txt` 가사 파일을 만듭니다
   - 예: `track1_song.mp3` → `track1_song.txt`
4. `script.js`의 `musicFiles` 배열을 MP3 파일명으로 업데이트합니다
5. 웹 브라우저에서 `index.html`을 엽니다

## 사용 방법

### 데스크톱

- **재생/일시정지**: 재생/일시정지 버튼을 클릭하거나 스페이스바 사용
- **이전/다음 트랙**: 네비게이션 버튼을 클릭하거나 키보드 단축키 사용
- **구간 이동**: 진행 바를 드래그
- **볼륨**: 볼륨 슬라이더로 조절
- **가사 탐색**: ↑ (위) 및 ↓ (아래) 화살표 키를 사용하여 가사 탐색
- **트랙 선택**: 하단의 리모컨 플레이리스트에서 트랙 클릭

### 모바일

- 모든 컨트롤이 터치에 최적화되어 있습니다
- 플레이어 크기가 모바일 화면에 맞게 최소화됩니다
- 리모컨 플레이리스트는 한 줄에 2개 항목을 표시합니다
- 가사 패널에는 리모컨과 겹치지 않도록 하단 패딩이 있습니다

## 커스터마이징

### 음악 파일 추가

`script.js`의 `musicFiles` 배열을 편집합니다:

```javascript
const musicFiles = [
    "track1_your_song.mp3",
    "track2_another_song.mp3",
    // 더 많은 트랙 추가...
];
```

### 스타일 수정

- `styles.css`를 수정하여 색상, 크기, 애니메이션 변경
- 미디어 쿼리의 중단점을 조정하여 다양한 화면 크기에 맞춤
- `@keyframes` 규칙에서 애니메이션 속도 및 효과 커스터마이징

### 가사 형식

가사 파일은 일반 텍스트(`.txt`) 형식이어야 합니다. 각 줄은 가사 한 줄을 나타냅니다. 플레이어는 자동으로:

- 가사를 줄로 분할
- 한 번에 3줄 표시
- 활성 줄을 중앙 정렬
- 탐색 시 부드러운 전환 효과 적용

## 브라우저 호환성

- 최신 브라우저 (Chrome, Firefox, Safari, Edge)
- JavaScript 활성화 필요
- 외부 의존성 없음

## 기능 상세 설명

### 레코드 판 애니메이션

- 재생 중 지속적으로 회전
- 시각적 매력을 위해 위아래로 바운스
- 음악이 일시정지되면 회전도 일시정지

### 가사 표시

- 한 번에 3줄의 가사 표시
- 활성 줄이 강조되고 더 크게 표시
- 부드러운 페이드인/페이드아웃 전환
- 화살표 키로 수동 탐색

### 리모컨 플레이리스트

- 하단의 가로 레이아웃
- 트랙 번호와 제목 표시
- 현재 재생 중인 트랙 강조
- 반응형: 모바일에서 한 줄에 2개 항목

### 배경 장식

- 다양한 애니메이션이 있는 떠다니는 구체
- 회전하는 링
- 여러 애니메이션 효과 (펄스, 드리프트, 글로우)
